About the Project
This project was created as part of my Machine Learning internship. The goal was to build a model that can predict whether a passenger survived the Titanic disaster based on information such as age, gender, passenger class, and fare.
Problem Statement
The Titanic dataset contains information about passengers who were on board the Titanic. Using this data, I trained a machine learning model to predict survival outcomes.
Tools and Technologies
Python
Pandas
Scikit-learn
Data Preparation
Before training the model, the dataset was cleaned and prepared by:
Selecting relevant features
Handling missing age values
Converting categorical data into numerical values
Splitting the dataset into training and testing data
Model Used
For this project, I used the Logistic Regression algorithm because it is simple, effective, and suitable for binary classification problems.
Model Performance
The model achieved an accuracy of approximately 80.45% on the test dataset.
Learning Outcomes
Through this project, I learned:
Data preprocessing techniques
Feature selection
Train-test splitting
Building classification models
Evaluating model performance using accuracy and classification reports
Conclusion
This project demonstrates the basic workflow of a machine learning classification problem, starting from data preprocessing to model evaluation. The results show that the model can predict passenger survival with good accuracy
