# AI Resume Analyzer

## Overview
AI Resume Analyzer is a Streamlit-based application that analyzes resumes and provides insights such as ATS score, detected skills, missing skills, and resume feedback.

## Features
- Resume PDF Upload
- Resume Text Extraction
- Skill Detection
- ATS Score Calculation
- Missing Skills Analysis
- Resume Feedback
- Job Description Matching

## Technologies Used
- Python
- Streamlit
- Pandas
- PyPDF
